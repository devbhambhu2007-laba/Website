A block theme for standalone blogs and social-media-aware publishing sites with open-social starter patterns.

== Description ==

This self-hosted WordPress block theme is built for bloggers, personal sites,
publications, and communities that want an editable publishing home. It can work
as a standalone blog, or as the home base for a site that also points readers to
social media profiles, feeds, reply paths, and open social networks.

It includes templates, template parts, style variations, and starter patterns for
explaining where work starts, how readers can follow it, and which social or
open channels a site owner chooses to use.

The theme provides templates, styles, and starter copy only; it does not publish
to social networks, add protocol endpoints, or require a plugin. Site owners can
add plugins, services, or manual links if they want deeper social publishing
integrations.

The theme includes Light and Dark style variations, plus an optional front-end
toggle that stores an explicit visitor choice in localStorage under the
reclaim-the-web-theme key. When no choice has been made, the theme follows the
visitor's system color preference.

== Privacy ==

This theme makes no tracking calls, loads no remote assets, and sends no
visitor data to third-party services. The theme does not install, download, or
recommend plugins. The only browser storage used by the theme is the optional
color-mode toggle, which stores the visitor's local display preference in that
visitor's browser.

== Installation ==

1. Upload the `reclaim-the-web` folder to `wp-content/themes`.
2. Activate the theme from Appearance > Themes.
3. Edit the templates, template parts, Styles, and starter patterns in the Site
   Editor.
